Much of the coding standards that we used for our coding of Javascript inside of our AngularJS
is from this web page, such as naming standards, project structure, a
https://github.com/johnpapa/angular-styleguide/blob/master/a1/README.md

Some things we did differently compared to this style:

Used $scope to define scope objects to keep it clear what is involved with the scope.

Using {{}} to define AngularJS variables inside HTML to keep everything consistent.
